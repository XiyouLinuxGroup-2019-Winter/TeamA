### 链表要求
##### 基本要求
- 1.链表的创建
- 2.增加(头插，尾插两个函数)
- 3.删除(删除第i个节点，删除所有值为key的节点两个删除函数)
- 4.修改节点信息
- 5.查找(查找第i个节点，查找值为key的节点两个删除函数)
##### 进阶要求
- 1.升级为循环链表
- 2.双向链表
- 3.双向循环链表
